# Jabulile I. Lubisi - Researcher Profile

A professional Streamlit web application showcasing research work in Data Science and Cybersecurity.

## Setup Instructions

### Adding Your Profile Photo

1. Save your profile photo as `profile.jpg`
2. Place it in: `streamlit_files/streamlit_files/assets/profile.jpg`
3. The app will automatically use your photo!

### Running Locally

```bash
cd "c:\Users\JB\OneDrive - University of Mpumalanga\Documents\Streamlit Assignment"
streamlit run streamlit_files/streamlit_files/app.py
```

### Deploying to Streamlit Cloud

1. Create a GitHub repository named `css2026-researcher-profile`
2. Push all files to the repository
3. Go to [share.streamlit.io](https://share.streamlit.io)
4. Connect your GitHub account
5. Select your repository
6. Set main file path: `streamlit_files/streamlit_files/app.py`
7. Customize URL to start with `css2026-`
8. Deploy!

Your public URL will be: `https://css2026-researcher-profile.streamlit.app`
